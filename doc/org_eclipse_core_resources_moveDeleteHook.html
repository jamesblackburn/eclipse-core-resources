Move/Delete Hook


Identifier: org.eclipse.core.resources.moveDeleteHook

Description: For providing an implementation of an IMoveDeleteHook to be used in the IResource.move and IResource.delete mechanism. This extension point tolerates at most one extension.

This extension point was added in Eclipse 2.0.

Configuration Markup:

   <!ELEMENT moveDeleteHook EMPTY>
   <!ATTLIST moveDeleteHook class CDATA #REQUIRED>

  • class - a fully qualified name of a class which implements org.eclipse.core.resources.IMoveDeleteHook.
Examples:

The following is an example of using the moveDeleteHook extension point.

(in file plugin.xml)

   <extension point="org.eclipse.core.resources.moveDeleteHook">
      <moveDeleteHook class="org.eclipse.team.internal.MoveDeleteHook"/>
   </extension>

API Information: The value of the class attribute must represent an implementation of org.eclipse.core.resources.IMoveDeleteHook.

Supplied Implementation: The Team component will generally provide the implementation of the move/delete hook. The extension point should not be used by any other clients.
